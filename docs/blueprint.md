# **App Name**: DataStream Dashboard

## Core Features:

- Real-time Data Visualization: Display large-scale data streams in real time using various chart types (line, bar, scatter, heatmap).
- Interactive Data Controls: Provide interactive controls for zooming, panning, filtering, and time range selection to explore the data.
- Data Aggregation: Allow users to aggregate data by different time periods (1min, 5min, 1hour) for better analysis.
- Performance Monitoring: Monitor FPS and memory usage to ensure the dashboard maintains optimal performance.
- Virtual Scrolling: Implement virtual scrolling for large datasets to improve rendering efficiency.

## Style Guidelines:

- Primary color: Deep teal (#16A385) to evoke a sense of stability and clarity, inspired by the serious application of visualizing complex datasets.
- Background color: Light teal (#E0F2F1) to ensure the focus remains on the data while providing a calm backdrop.
- Accent color: Light Green (#86EFAC) to draw attention to interactive elements and key data points.
- Body and headline font: 'Inter' for a modern, neutral, and easily readable appearance. 
- Use minimalist icons to represent different data types and control options.
- Design a clean, modular layout that adapts to different screen sizes and resolutions, ensuring a consistent experience across devices.
- Implement smooth transitions and animations for data updates and user interactions to enhance usability.
